Running tasks:
To run the tasks navigate to the "src" directory by running "cd cs4300/homework1/src" and run "python3 task<task number>.py" in the terminal.
For task7, go into the environment and run "pip install numpy" before running.

Running tests:
To run tests on all tasks navigate to the "homework1" directory by running "cd cs4300/homework1" and run "pytest" in the terminal.
For task7, go into the environment and run "pip install numpy" before running pytest.
